  <files>layouts/index.html</files>
  <action>
In layouts/index.html, locate the reviews rendering logic (around lines 348-355).

Current broken code (appends fallbacks unconditionally):
```hugo
{{ $reviewsData := .Site.Data.reviews | default dict }}
{{ $displayReviews := slice }}
{{ range $reviewsData.reviews }}
    {{ $displayReviews = $displayReviews | append . }}
{{ end }}
{{ range $reviewsData.fallbackReviews }}
    {{ $displayReviews = $displayReviews | append . }}
{{ end }}
```

Replace with conditional logic (switches to fallback only when live is empty):
```hugo
{{ $reviewsData := .Site.Data.reviews | default dict }}
{{ $displayReviews := slice }}
{{ if gt (len ($reviewsData.reviews | default slice)) 0 }}
    {{ range $reviewsData.reviews }}
        {{ $displayReviews = $displayReviews | append . }}
    {{ end }}
{{ else }}
    {{ range $reviewsData.fallbackReviews }}
        {{ $displayReviews = $displayReviews | append . }}
    {{ end }}
{{ end }}
```

Key change: Use fallbackReviews ONLY when the live reviews array is empty, not in addition to live reviews.
