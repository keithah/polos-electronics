<decisions_made>

- Phase 7 will be a new phase (post v1.1) for interactive mapping; not part of Phase 4.
- Interactive map implementation: Google My Maps iframe embed (no paid Google Maps JS API).
- Reliability requirement: keep static SVG/image as fallback; Service Area section must always show map context.
- Provide an escape hatch link: "Open in Google Maps".

Related Phase 4 decisions carried forward:
- Do not fabricate reviews. Only show real reviews; if ingestion is empty, show last-known-good fallback reviews.
